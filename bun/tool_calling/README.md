To install dependencies:

To test:
```shell
bun test
```

To run:
```shell
bun run main.ts -t "Add 200 and 1"  # triggers add numbers tool
bun run main.ts -t "Make a note 'Buy new coffee'"  # triggers make note tool
```
